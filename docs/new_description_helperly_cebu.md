Helperlyは、フィリピン・セブ島を拠点とする
生活支援サービスのマッチングプラットフォームです。

清掃、家事代行、ベビーシッターなどのサービスを必要とする利用者と、
それらのサービスを提供するフィリピン人ワーカーを、
モバイルアプリ上でつなぎます。

利用者は、日時やサービス内容を指定してワーカーを検索し、
プロフィールやレビューを確認したうえで予約を行います。
予約、チャット、決済までをアプリ内で完結でき、
言語は英語・日本語・タガログ語・セブアノ語に対応しています。

ワーカー側は、提供可能なサービス、時給、稼働可能な時間帯を登録し、
予約リクエストを受けて仕事を行います。
自分で働く時間と報酬を設定できるため、柔軟な就労が可能です。

収益モデルはマーケットプレイス型で、
ワーカーが設定した時給に対してプラットフォーム手数料（20%）を上乗せした金額を
利用者に請求します。
決済はオンラインで行われ、プラットフォーム手数料が収益となります。

現在はセブ島を中心に、
外国人居住者向けの清掃・家事代行サービスから事業を開始しており、
今後はサービスカテゴリの拡張や、
他都市への展開を予定しています。
