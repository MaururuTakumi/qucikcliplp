# ② 事業紹介（詳説版）

## 1. 私たちのミッション

**日本のプロダクト開発に、世界級の若手エンジニアを。**
インドの優秀な新卒エンジニアが持つ、**CS基礎・実務経験・英語力**を、日本のものづくりに接続します。

---

## 2. 当社のアプローチ（3つの土台）

**(1) 現地ネットワーク × 学生コミュニティ**

* エンジニア学生団体との連携で、**質の高い母集団**を安定確保
* Redditの学生コミュニティで当社投稿が**30,000インプレッション／日別15位**到達
* オンラインに偏らず**オフライン接点**（勉強会・ミートアップ）も設計

**(2) Tierに依らないスキル評価**

* IITs等の大学名ではなく、**実力ベース**で評価
* **コーディングテスト／ワークサンプル／設計レビュー**で多面的にチェック
* **英語面接**で業務コミュニケーション力も確認

**(3) フェアで適正なコスト設計**

* 「安さ」ではなく、**国際市場基準の適正水準**でのご提案
* 採用後の**活躍と定着**まで見据えた**総コスト最適化**（採用・オンボーディング・教育）

---

## 4. 採用課題への解決策：AIスクリーニング

**課題**
「外国人向け求人を出した瞬間、**大量応募**で見切れない」

**解決**

* 応募集約（HP／LinkedIn／CSV）→ **1クリックAIスクリーニング**
* 例：`Name | TypeScript | Python | Database | Culture Fit | Contact`

  * 技術：**Best / Enough / Not Bad**
  * Culture Fit：**スコア（0–100）**
* **コード意図の口頭説明**で**生成AIコピペ対策**
* 根拠の**可視化（説明責任）**で社内合意形成もスムーズ

**導入効果（指標イメージ）**

* スクリーニング時間 **70–90%削減**
* **一次面接通過率の標準化**、早期離職の低減
  ※数値は導入企業の実測に応じて更新
