### デザイン言語
- **配色**: 藍/紫を主色(~#4C37F5)、薄い青紫(#A5B4FC/#C7D2FE)、ほぼ白〜淡グレー背景(#F4F4F6)、ほぼ黒の文字。採用セクションだけ濃紺(#1E2530)で切り替え
- **タイポ**: クリーンな幾何学ゴシック。見出し特大。**日本語ヒーローは縦書き**、英字をアクセントラベルに
- **余白**: たっぷり。テキスト密度は低く、縦のリズムが広い
- **モーションが署名**: スクロール連動3D／斜めのウェッジ・トランジション／円形の矢印ボタン／フローティングナビ

## 4. Fable5でLPを作る「型」— 4ステップ（tatsuki氏記事の要約）

出典: [Fable5で高品質LPを作る４ステップ](https://x.com/nobel_824/status/2073269533528449089)（@nobel_824, 28万Views/1.4Kブックマーク）
**この順番を守る。特にStep1を飛ばすと何回直しても"AIっぽさ"から抜けられない。**

- **環境**: Claude Code（デスクトップアプリで可）＋モデルにFable 5。思考量(effort)は「**最初の一手だけmax、以降medium、デザインシステム構築はhigh**」で使い分け
- **Step1: デザインを"先に"固める** — いきなり「かっこいいLP作って」と言わない。土台(デザインシステム/配色/フォント/禁止リスト)を先に宣言。Anthropic製「Frontend Design」スキル＋「UI/UX Pro Max」プラグインが効く。← ここがAIっぽさから抜ける分岐点
- **Step2: 参照を集める（言葉より実物）** — サイトを丸ごと真似ない。良いサイトから**セクション単位**で切り取り(「このヒーローだけ」等)、参照フォルダに放り込む。← 今回はLayerXがこれ
- **Step3: 一発で骨格を出す（マスタープロンプト）** — 参照＋狙いを渡す。プロンプト末尾に必ず「**作る前にスタイル/フォント/セクション構成/アニメの強さ/文章トーンを質問して**」を足す→ AIが4〜6問返す→答えが土台になる
- **Step4: 磨く（飛ばす人が損）** — 直しは一つずつ投げず「こことここ、まとめて直して」。仕上げにAI自身へ品質チェック（フォント使い回し/配色絞れ/文字サイズの視線誘導/アニメの意図/モバイル設計/AIっぽい埋め草）を依頼
- **+コツ**: 「終わるまで続けて」で完走を促す／音声指示も有効
- **3Dの種明かし**: 多くは本物の3Dでなく「AIで短い動画を1本作り、スクロール位置に合わせ再生/逆再生」。効かせるコツ=「1セクション1エフェクト」「派手すぎたら"控えめに洗練"と押し返す」「モバイルは明示的に整えさせる」
- **釘刺し**: 一発で完璧は出ない(2〜3回の磨き前提)／他人のプロンプト丸コピーは伸びない(独自性は自分で足す)／Fable5の料金形態は流動的、使う前に確認

## 9. アニメ制作の技術方針 と MCP動画ツールの要否

**結論: LayerX級のモーションは9割コードで作れる。higgsfield/topview の接続は"必須ではない"。**

- **コードで完結（推奨・第一手）**: §7の #1,3,4,5,6,7,8,9 は Framer Motion（既に依存に有）で実装可。#2の3Dは **React Three Fiber(Three.js)** を足せば幾何形状（キューブ/ディスク/球）はコードで生成・スクロール連動できる。LayerXのビジュアルは幾何学的・抽象的なので、実写/生成動画より **コード3Dの方が軽く・保守しやすく・忠実**。
- **既接続の nanobanana(Gemini画像)** で足りる用途: ヒーロー用の抽象アート・テクスチャ・背景静止画・OG画像。まず静止画はこれで賄える。
- **AI動画生成(higgsfield/topview)が要るのは限定ケースのみ**:
  - **higgsfield**（画像→動画・カメラワーク）: もし「スクロールでスクラブする短い実写風/有機的ヒーロー動画」路線に決めた場合に有用。geometric路線なら不要。
  - **topview**（アバター/TTS/マーケ動画。skill有り）: 用途がアバター・宣伝動画寄りで、LayerX風コーポレートHeroとは方向が違う。**今回の再現には不要**。
- **推奨判断**: まずは **Framer Motion + React Three Fiber + nanobanana静止画** で組む。ヒーローに「コードでは出しにくい有機的な映像」を入れたいと fableとの議論で決まったら、その時に **higgsfield** をMCP接続する（topviewは今回スコープ外）。
- ※ 接続が必要になったら、higgsfield MCP の登録情報（サーバURL/認証）を教えてください。こちらからは接続情報を持っていません。
