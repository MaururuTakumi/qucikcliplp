アカスタ（Akamon Study）事業概要（コーポレートサイト用）

アカスタ（Akamon Study）は、中学受験を中心とした学習ニーズを持つ家庭と、東大生をはじめとする優秀な大学生講師を、必要なときにスポットでつなぐ家庭教師マッチングサービスです。

従来の家庭教師サービスでは、月額契約や長期契約が前提となるケースが多く、
「テスト前だけ見てほしい」「苦手単元を短期間で対策したい」といったニーズに十分対応できていないという課題がありました。

アカスタはこの課題に対し、
高品質な講師に、1回単位・短期間から依頼できる仕組みを提供することで、
より柔軟で使いやすい学習体験を実現しています。

講師は運営による審査・管理のもとで登録され、授業のスケジュール管理、予約、決済、授業実施までを一貫してプラットフォーム上で完結させています。
授業はオンラインを基本とし、忙しい家庭でも利用しやすい設計となっています。

アカスタは、
「必要な学習を、必要なタイミングで、信頼できる人から受けられる」
新しい家庭教師の形を提供することで、家庭と講師の双方にとって持続可能な学習プラットフォームを目指しています。
